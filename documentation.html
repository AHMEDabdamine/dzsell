  1. 1. Download the Prerequisites

    Download the required files from our website. These will include the following:
    • wamp server installer
    • Microsoft C++ package installer

  2. 2. Purchase Dz SELL and Download

    Purchase Dz SELL from our website or contact us and obtain the download link. Download the Dz SELL package.

  3. 3. Extract the Prerequisites

    Extract the contents of the downloaded "prerequisites.zip" file. You will find two files:

    • wamp server installer
    • Microsoft C++ package installer

  4. 4. Install Microsoft C++ Package

    Extract the "visual studio C++ pack" and locate the "install all .bat" file. Execute this file and wait for the installation to complete. Accept all prompts during the installation. Don't worry if you see a command prompt, this is normal.

  5. 5. Install WAMP Server

    Install the WAMP server by executing the downloaded .exe installer. Follow these steps:

    1. Choose your language and click "OK".
    2. Accept the terms of usage and click "Next".
    3. Choose the location where you want to save the WAMP files. Remember this location and click "Next".
    4. Continue clicking "Next" until the WAMP server is installed.

  6. 6. Install Dz SELL

    Execute the Dz SELL installer. A window will pop up. Click "OK". The installer will prompt you for a location to save the Dz SELL files. Install it in the "www" folder of your WAMP server (e.g., C:\wamp64\www\Dz SELL). If you are installing it on the C drive, you can just click "Next" as the default path should be correct.

  7. 7. Start WAMP Server

    Go to the WAMP shortcut you just installed and run it. Wait for all services to start. You will see a green 'W' icon on your taskbar.

  8. 8. Configure PHP Version

    Click on the green "W" icon on the taskbar. Navigate to "PHP" and change the PHP version to 7.3. The server will restart automatically.

  9. 9. Access phpMyAdmin

    Click on the green "W" icon again. Navigate to "phpMyAdmin" and run it. A new tab will open in your web browser and phpMyAdmin will start.

  10. 10. Create the Dz SELL Database

    Enter the username "root" (without a password). In the "Databases" section, click "New" to create a new database. Name it "dzsell". This will create a database named "dzsell".

  11. 11. Import the Database

    Click on "Import" and then "Browse". Go to this path: C:\wamp64\www\Dz SELL\databases\dzsell.sql. Scroll down and click "Import". Wait for a few seconds, and you will see a confirmation message.

  12. 12. Create a Virtual Host

    Go back to the green "W" icon on the taskbar. Navigate to "localhost" (or type "localhost" in your browser). This will open a page with localhost controls. Click "Add New Virtual Host".

  13. 13. Configure Virtual Host

    Fill in the following fields:

    • **First Input:** Enter "www.dzsell.com" or any other desired name (use a short name like "dzsell" or "pos").
    • **Second Field:** Enter the path where you installed Dz SELL (e.g., C:\wamp64\www\Dz SELL).
    • **Third Field:** Leave this empty.

    Click "Create" to create the virtual host. Restart the WAMP server by right-clicking the green "W" icon and selecting "Restart all services".

Dz SELL Interface

Dz SELL has 9 modules: Customers, Items, Item-Kits, Suppliers, Reports, Receivings, Sales, Expenses, and Office. Each module serves a specific function in managing your business.

Each Module Function

  • 1. Customers Module

    Streamline your customer management. This module facilitates quick registration, allowing you to easily track customer details for efficient sales operations. You can also temporarily suspend sales for specific customers as needed.

  • 2. Items Module

    Maintain a comprehensive inventory. This module allows you to manage your store's stock, adding and updating items with ease. The built-in barcode generator helps you create unique labels for your products, simplifying inventory tracking and management.

  • 3. Item Kits Module

    Offer bundled deals. This module empowers you to create pre-defined packages of items, making it simple to sell popular combinations and boost sales.

  • 4. Suppliers Module

    Organize your supplier relationships. This module provides a centralized platform to manage your suppliers, assigning them unique IDs and tracking their sales. It simplifies communication and ensures a smooth flow of inventory.

  • 5. Reports Module

    Gain valuable insights. This module provides powerful reporting capabilities, offering detailed insights into your sales, inventory, and other key metrics. With its comprehensive reporting features and visualizations, you can make data-driven decisions for your business.

  • 6. Receivings Module

    Manage incoming inventory. This module streamlines the receiving process, allowing you to track incoming items, manage supplier deliveries, and ensure accurate inventory updates.

  • 7. Sales Module

    Drive your sales. This is the heart of the system, equipped with features to simplify and enhance your sales process. It includes functionality for suspending items, automating receipt and invoice generation, and sending customer emails, streamlining your sales operations.

  • 8. Expenses Module

    Track your business expenses. This module helps you effectively manage and categorize your store's expenses, from utilities like electricity and fuel to other business-related expenditures. This provides you with a clear understanding of your financial health.

  • 9. Office Module

    Centralized configuration and management. This module is your command center, offering the ability to configure and manage various system settings, including employee details, permissions, and other critical aspects of your business operations.
